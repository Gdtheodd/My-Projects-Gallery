Investigating Titanic Dataset¶

In this document I will include some analysis for the titanc data file using Pandas. the file contains some information of 891 passengers and crew on board of the titanic.

By analysing the data, I will try to find answers for the following questions:

  • what was the survival rate in the given dataset?
  • what are the characterstics of the survived?
  • was passengers of class 1 are more likley to survive?

As a conclusion of the above analysis:¶

female passengers in this dataset were more likley to survive . passengers in the upper class (Pclass=1) had a better chance to survive. while the number of survived children in class 3 was higher than the other 2 classes.

One of the question I was interested to invistigate is what is the crew survival rate? but to limitation in the data and missing cabin information I could not conclude which male passenger is of the crew.
